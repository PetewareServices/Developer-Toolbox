# Developer Toolbox | Peteware

---

## Overview

The Developer Toolbox is a versatile Roblox tool built to help anti-cheat developers efficiently build, debug, and test their anti-cheats. It offers a set of essential utilities to speed up development workflows and enhance security testing.

This project is fully open source, allowing anyone to access, modify, and contribute to the code. Community collaboration helps maintain consistency and reliability across the toolbox.

---

## Features

- **Quick Launch for Popular Tools**  
  Includes buttons for well-known utilities like Infinite Yield, Remote Spy, Dex Explorer, and more.

- **Instance Scanner**  
  Search game instances by class name to locate and inspect specific elements within your game.

- **Addons Manager**  
  Add, manage and execute your own scripts from the addons manager.

- **Server Hop & Rejoin Buttons**  
  Quickly switch or rejoin servers.

- **Executor Info**
  Prints useful executor information such as executor level and context.

- **Queue on Teleport**  
  Automatically loads the Developer Toolbox whenever you rejoin or switch servers, keeping your tools ready.

---

## Usage

Paste this script into your Roblox executor and execute

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/PetewareServices/Developer-Toolbox/refs/heads/main/main.lua"))({["Owner"] = "PetewareServices"})
```
